### 📂 How to Add Custom Sounds

This directory acts as the storage container for **YChang Sound** system sounds. You can personalize your workflow by adding any of these supported formats:
* 🎧 `.mp3`
* 🎧 `.wav`
* 🎧 `.ogg`

### ⚙️ Quick Setup Guide

1. **Drop your file:** Move your desired audio file directly into this folder. Let's call it `my-awesome-sound.mp3`.
2. **Update Settings:** Open your VS Code settings, search for `YChang Sound`, and change the `ychang-sound.soundFile` value to `my-awesome-sound.mp3`.
3. **Enjoy!**

> **Note:** The extension is pre-configured to automatically look for a file named `error.mp3`. If you want to use the default configuration, just name your file `error.mp3`!
